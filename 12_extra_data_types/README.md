# Extra Data Types

This pattern is extremely useful for scheduling systems, project management tools, automation platforms, and any application dealing with time-based operations.

We learned about multiple datatypes:

1. `datetime.datetime`
2. `datetime.date`
3. `datetime.time`
4. `datetime.timedelta`
5. `UUID`
6. `frozenset` - Convert list into set (to eliminate duplicates) then to list again
