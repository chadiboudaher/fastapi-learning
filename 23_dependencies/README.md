# Dependencies

## What is `Dependency Injection`?

It means that there are ways for your code to declare things things that it require to work and use: `dependencies`.

## What dependencies do?

They are reusable components that can be injected into your path operations. They handle common tasks like:

- Authentication/Authorization.
- Database sessions.
- Logging & monitoring
- Rate limiting
- caching
- COnfiguring loading
