# Middleware

A `Middleware` is a function that works with every request before it is processed by any specific path operation. and with every response before returning it.

- It takes each request that comes to your application.
- It can then do something to that request or run any needed code.
- Then it passes the request to be processed by the rest of the application (by some path operation).
- It then takes the response generated by the application (by some path operation).
- It can do something to that response or run any needed code.
- Then it returns the response.

## Create middleware

`@app.middleware("http")` on top of a function.
It receives:

1. The request
2. A function `call_next` that will receive the `request` as a parameter.
